# 审批回调

#### 1.接口请求地址:

测试环境域名:  http://ipaas.srm-dev.naura.com

接口地址: http://ipaas.srm-dev.naura.com/flow/api/cloud-srm/api-pj/flow/event/callBack

请求方式: POST

请求数据类型: application/json

响应数据类型: */*

接口描述: 审批回调

#### 2.请求header:

| ***\*参数名称\**** | ***\*参数说明\**** | ***\*类型\**** | ***\*值\****                                 | ***\*schema\**** |
| ------------------ | ------------------ | -------------- | -------------------------------------------- | ---------------- |
| username           | 用户名             | string         | ESB                                          | header           |
| password           | 密码               | object         | f2d2e572429780af465864421b7a1fba  (固定密码) | header           |

#### 3.请求参数:

| ***\*参数名称\****  | ***\*参数说明\**** | ***\*是否必须\**** | ***\*数据类型\**** | ***\*schema\**** |
| ------------------- | ------------------ | ------------------ | ------------------ | ---------------- |
| callBackType | 回调类型 ADD-审批新增 UPDATE-审批更新 REJECT-审批驳回 ABANDON-审批废弃 APPROVAL-审批通过 | Y | string ||
| templateId | 模板ID                                                       | N | string ||
| flowId | 流程ID | Y | string ||
| flowData | 流程数据(JSON格式) | N | string ||
| node | 节点 | N | string ||

#### 4.请求body:

```json
{
    "callBackType": "ADD",
    "templateId": "201208121638",
    "flowId": "202308141506",
    "node": "N8",
    "flowData": {
        "claimNumber": "202308121651",
        "materialList": [
            {
                "materialCode": "202308141605"
            }
        ]
    }
}
```

#### 5.响应状态码:

| ***\*状态码\**** | ***\*说明\**** | ***\*schema\**** |
| ---------------- | -------------- | ---------------- |
| 200              | OK             |                  |
| 201              | Created        |                  |
| 401              | Unauthorized   |                  |
| 403              | Forbidden      |                  |
| 404              | Not Found      |                  |
| 500              | System Error   |                  |

#### 6.返回参数

| ***\*参数名称\**** | ***\*参数说明\****        | ***\*类型\**** | ***\*schema\**** |
| ------------------ | ------------------------- | -------------- | ---------------- |
| code               | 响应码,成功0,其他均为失败 | string         |                  |
| data               | 响应结果                  | object         |                  |
| message            | 响应描述信息              | string         |                  |

#### 7.返回参数结构:

```json
{
    "code": "0",
    "data": null,
    "message": "操作成功"
}
```

#### 8.返回解释:

```
除了返回0以外,都是异常
```
